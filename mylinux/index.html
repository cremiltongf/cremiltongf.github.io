Configurações

Em Configurações do Fedora:

  1. Multitarefas → Geral
    1. Canto ativo - ativo;
    2. Bordas de tela ativas - desativado.
  2. Aparência → Estilo
    1. Escuro → selecione.
  3. Aparência → Cor de realce
    1. Azul → selecione.

Gnome Tweaks

Instalação


sudo dnf install gnome-tweaks

Configuração

  1. Aparência
    1. Estilos → Ícones → Kora.
  2. Janelas
    1. Botões da barra de título
      • Maximizar → desativado;
      • Minimizar → ativo;
      • Posicionamento → esquerda.
    2. Ações de clique
      • Centralizar novas janelas → ativo.

Gerenciador de Extensões

Instalação


sudo flatpak install flathub com.mattjakeman.ExtensionManager
Instale as extensões abaixo através do Gerenciador de Extensões.

AppIndicator

O AppIndicator and KStatusNotifierItem Support adiciona uma área ao painel para exibir ícones indicadores. Essão extensão não requer nenhuma configuração, mas é possível mudar o local de exibição por exemplo, que por padrão é a direita.

Mais detalhes: AppIndicator and KStatusNotifierItem Support.

O Fedora já traz instalado. Só precisa ativar.

Blur my Shell

Adiciona uma aparência desfocada a diferentes partes do GNOME Shell, incluindo o painel superior, o painel e a visão geral.

Mais detalhes: Blur my Shell.

Configuração

  1. Pipelines
    1. Adicionar Pipeline chamado "Visão Geral"
      • Adicionar efeito - Desfoque gaussiano nativo
        • Raio → 3
        • Brilho → 0,37
    2. Adicionar Pipeline chamado "Dash"
      • Adicionar efeito - Desfoque gaussiano nativo
        • Raio → 3
        • Brilho → 0,37
      • Adicionar efeito → Canto
        • Raio → 24
  2. Painel
    1. Tipo de desfoque → Dinâmico
    2. Sigma → 3
    3. Brilho → 0.37
    4. Substituir plano de fundo → ativado
      • Estilo de plano de fundo → transparente
      • Desativar quando uma janela está próxima → ativado
      • Desativar na visão geral → ativado
  3. Visão geral
    1. Pipeline "Visão Geral"
    2. Estilo dos compomenentes da visão geral → escuro
    3. Sigma → 30
    4. Brilho → 0,60
    5. Estilo de caixas de diálogo da pasta do aplicativo → escuro
  4. Dash
    1. Tipo de desfoque → estático
    2. Pipeline → "Dash"
    3. Substituir plano de fundo → Estilo de plano de fundo → transparente
    4. Desativar na visão geral → ativado

Dash to Dock

Um dock para o Gnome Shell.

Mais detalhes: Dash to Dock.

Configuração

  1. Posição e tamanho
    1. Exibir o dock → monitor primário
    2. Posição na tela → embaixo
    3. Ocultação inteligente → ativo
    4. Ocultação inteligente → configuração
      • Esconder janela → desativado
  2. Lançadores
    1. Mostrar lixeira → desativado
    2. Show the number of unread not... → desativado
  3. Comportamento
    1. Ação do clique → focar, minimizar ou mostrar pré-visualizações
    2. Ação da rolagem → não fazer nada
  4. Aparência
    1. Encolher o dash → desativado
    2. Mostrar visão geral ao inicializar → desativado
    3. Usar o tema do sistema → ativo
